How to Fix CSV Encoding Issues

Common Encoding Issues
Your CSV file contains special characters that aren't compatible with UTF-8 encoding. This happens when files are saved with different character encodings. Any CSV file format is accepted - no restrictions on delimiters, structure, or content.

📋 Detailed Step-by-Step Solutions

🔧 Method 1: Microsoft Excel (Recommended)
⭐ Best for: Most Windows users, large files, complex data
  1. Open Excel → File → Open → Select your CSV file
  2. File → Save As (or press F12)
  3. Choose location where you want to save the file
  4. File name: Give it a new name (e.g., "mydata_utf8.csv")
  5. Save as type: Select "CSV UTF-8 (Comma delimited) (*.csv)"
  6. Click Save
  7. Re-upload the new UTF-8 file to this system
💡 Tip: If you see a warning about features, click "Yes" to continue. The UTF-8 version will work perfectly.
🌐 Method 2: Google Sheets (Free & Easy)
⭐ Best for: Quick fixes, no software installation, any device
  1. Go to sheets.google.com
  2. File → Import → Upload → Select your CSV file
  3. Wait for upload (may take a moment for large files)
  4. File → Download → Comma-separated values (.csv)
  5. The downloaded file will automatically be UTF-8 encoded
  6. Re-upload this new file to our system
💡 Tip: Google Sheets handles encoding automatically - no technical settings needed!
📝 Method 3: Notepad++ (Advanced Users)
⭐ Best for: Technical users, precise control, very large files
  1. Download and install Notepad++ (free)
  2. Open Notepad++ → File → Open → Select your CSV
  3. Look at bottom status bar - you'll see current encoding (e.g., ANSI)
  4. File → Save As
  5. Encoding: Change from current to UTF-8
  6. Save the file with a new name
  7. Re-upload the UTF-8 version
💡 Tip: The encoding dropdown is near the bottom of the Save As dialog. Look for "Encoding" field.
⚡ Method 4: Quick Copy-Paste (Emergency Fix)
⚠️ Use only if: Other methods fail, file is small (<10,000 rows)
  1. Open CSV in Excel or any spreadsheet program
  2. Select all data (Ctrl+A or Cmd+A)
  3. Copy (Ctrl+C or Cmd+C)
  4. Open new blank workbook
  5. Paste data (Ctrl+V or Cmd+V)
  6. Save As → CSV UTF-8 (see Method 1)
✅ Quick Tip
If you're still having issues, try copying the data from your CSV and pasting it into a new spreadsheet, then save as CSV UTF-8.
